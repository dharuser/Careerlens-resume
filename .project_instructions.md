# CareerLens – AI Resume Analyzer & Career Advisor

## Overview
Full-stack NextJS app where users upload PDF resumes and get AI-powered analysis including score, strengths, weaknesses, missing skills, career paths, job matches, and improvement tips.

## Tech Stack
- NextJS 14 + React 18 + Tailwind CSS 3
- PostgreSQL (Abacus AI hosted) via Prisma ORM
- Abacus AI LLM API (gpt-5.4-mini) for resume analysis
- AWS S3 for PDF storage
- HTML-to-PDF API for report download

## Theme & Design
- **Dark theme only** (no light mode toggle)
- Background: `#0F0F1A` (HSL 240 20% 6%)
- Cards: `#1A1A2E` (HSL 240 20% 11%)
- Primary accent: Purple `#7C3AED` (HSL 262 83% 58%)
- Secondary accent: Blue `#3B82F6` (HSL 217 91% 60%)
- Gradient: purple-to-blue for CTAs and accents
- Fonts: DM Sans (body), Plus Jakarta Sans (display), JetBrains Mono (numeric data)
- Framer Motion animations throughout

## Pages
1. **/** - Landing page with hero section, tagline, CTA, and 3 feature cards
2. **/analyze** - Upload PDF + target role input → streaming AI analysis → results dashboard
3. **/history** - Past analyses list from database

## Database Schema (Prisma)
- `Analysis` model: id, fileName, targetRole, score, strengths (JSON), weaknesses (JSON), missingSkills (JSON), careerPaths (JSON), jobMatches (JSON), tips (JSON), cloudStoragePath, isPublic, createdAt

## API Routes
- `POST /api/analyze` - Streams AI analysis (SSE), saves to DB
- `GET /api/history` - Returns past analyses
- `POST /api/download-pdf` - Generates PDF report via HTML-to-PDF API
- `POST /api/upload/presigned` - Gets S3 presigned upload URL

## Key Patterns
- PDF sent to LLM as base64-encoded file (no pdf-parse)
- Streaming SSE from API route to client with progress updates
- JSON response_format for structured LLM output
- S3 presigned URLs for file uploads (private, not public)
- Optional chaining everywhere for crash prevention
